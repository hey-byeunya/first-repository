# 우리 동네 중고거래 키워드 알림 — 서비스 기획서

## 배경 및 문제

사용자들이 원하는 조건의 중고 물품이 올라와도 실시간으로 확인하기 어려워, 좋은 매물을 놓치고 이탈하는 경우가 많다. 매번 앱을 열어 검색해야 하는 번거로움이 재방문을 가로막고 있다.

## 목적 및 목표

키워드 등록만으로 원하는 매물을 놓치지 않게 해 재방문율을 높인다. 3개월 내 알림 클릭률 15%, 알림을 통한 거래 성사 건수 월 500건 달성을 목표로 한다.

## 타겟 사용자

특정 키워드의 중고 물품을 반복적으로 검색하는 활성 사용자. 특히 육아용품, 가전처럼 시세가 자주 바뀌고 매물이 빨리 소진되는 카테고리를 찾는 사용자.

## 핵심 기능

원하는 키워드를 등록하면, 조건에 맞는 매물이 새로 올라올 때마다 푸시 알림을 보낸다.

## 사용자 시나리오

1. 검색 화면에서 원하는 키워드를 등록한다
2. 조건에 맞는 매물이 올라오면 즉시 푸시 알림이 온다
3. 알림을 누르면 매물 상세 화면으로 이동한다
4. 마음에 들면 채팅으로 바로 연락한다

## 화면 요소

- **키워드 등록 화면** — 알림받을 키워드를 추가하거나 삭제한다
- **알림함** — 도착한 알림을 최신순으로 모아 보여준다
- **매물 상세 화면** — 알림을 누르면 이동하는 화면으로, 사진과 가격, 채팅 버튼을 보여준다

## 성공 지표

키워드 알림 클릭률, 알림을 통한 채팅 시작 건수, 알림을 통한 거래 성사 건수

## 예외 상황

| 상황 | 동작 |
|------|------|
| 등록한 키워드가 20개를 넘으면? | 더 이상 추가할 수 없다는 안내를 표시한다 |
| 알림 권한이 꺼져있으면? | 인앱 배지로만 알림을 보여준다 |
| 같은 매물에 대해 알림이 중복으로 발송되면? | 첫 알림만 보내고 이후 중복은 발송하지 않는다 |

## 이번 범위에서 제외

- 키워드 알림의 우선순위 설정 기능은 이번 범위에서 제외한다
- 알림 도착 시간대를 사용자가 직접 설정하는 기능은 이번 범위에서 제외한다
