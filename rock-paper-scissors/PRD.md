# 가위바위보 — 게임 기획서

## 무엇을

컴퓨터(AI)를 상대로 가위·바위·보를 겨루는 3판 2선승제 웹 게임.

## 누가

웹 브라우저를 쓰는 누구나. 설치 없이 링크만 열면 바로 플레이할 수 있다.

## 규칙과 끝나는 조건

### 기본 규칙

- 플레이어가 가위·바위·보 중 하나를 선택하면, 컴퓨터도 무작위로 하나를 선택한다.
- 선택 후 가운데에 **3 → 2 → 1 → 0 카운트다운**이 표시되고, 0에서 양쪽의 선택이 **동시에** 공개된다.
- 가위는 보를 이기고, 바위는 가위를 이기고, 보는 바위를 이긴다.
- 같은 것을 내면 무승부이며, 무승부는 승수에 포함되지 않는다.

### 끝나는 조건

**3판 2선승제.** 플레이어 또는 컴퓨터 중 먼저 2승을 거두는 쪽이 나오면 게임이 종료된다.

## 화면 요소

### 플레이 화면

- **스코어보드** — 상단 중앙에 "나 : 컴퓨터" 점수 표시 (예: 1 : 0)
- **아레나** — 가운데 좌우로 플레이어(나)와 컴퓨터(COM)의 선택을 보여주는 카드 2개, 사이에 "VS" 텍스트
- **카운트다운** — 선택 후 아레나 가운데에 3-2-1-0 숫자가 크게 표시되며, 카운트다운 중 양쪽 카드의 이모지는 흐리게 처리
- **라운드 결과** — 아레나 아래에 승리!/패배/무승부 텍스트 표시 (승리는 초록, 패배는 빨강, 무승부는 회색)
- **선택 버튼** — 하단에 가위(✌️)·바위(✊)·보(🖐️) 버튼 3개
- **카드 하이라이트** — 결과에 따라 이긴 쪽은 초록, 진 쪽은 빨강, 무승부는 회색 테두리와 배경

### 게임 오버 화면

- 블러 처리된 오버레이 위에 결과 표시
- **승리했습니다!** (초록) 또는 **패배했습니다** (빨강) 텍스트
- 최종 스코어 (예: 2 : 1)
- **다시하기** 버튼 — 누르면 점수·상태 모두 초기화되고 새 게임 시작

## 예외 상황

| 상황 | 동작 |
|------|------|
| 카운트다운 애니메이션 도중에 버튼을 누르면? | 무시한다. 진행 중인 라운드가 끝날 때까지 버튼이 비활성화된다 |
| 이미 한쪽이 2승을 거둬 게임이 끝난 상태에서 선택 버튼을 누르면? | 아무 반응도 하지 않는다. 다시하기 버튼만 동작한다 |
| 같은 라운드에서 버튼을 여러 번 빠르게 클릭하면? | 처음 클릭한 선택만 반영된다. 이후 클릭은 모두 무시 |
